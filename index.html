WWEPhotos

Pull Photos from WWE Gallery Pages

⚠️ Browser Extension Required Due to CORS restrictions, you'll need to install a CORS extension to use this tool:
• Chrome: Allow CORS
• Enable it before clicking the download button
How to use:
1. Install and enable the CORS extension (see warning above)
2. Paste a WWE gallery URL
3. Click "Fetch Images" to preview
4. Click "Download All" to save them to your Downloads folder
